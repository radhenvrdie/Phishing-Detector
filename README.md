# 🛡️ Phishing Website Detector

## 📌 Description
This project detects whether a website is safe or phishing using URL analysis techniques.

## 🚀 Features
- Detect suspicious URLs
- Fast and simple interface
- Real-time result (Safe / Dangerous)
- User-friendly design

## 🛠️ Technologies Used
- Node.js
- Express.js
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/radhenvrdie/Phishing-Detector.git

2. Navigate to project folder:
   cd Phishing-Detector

3. Install dependencies:
   npm install

4. Run the server:
   node server.js

5. Open browser:
   http://localhost:3000
